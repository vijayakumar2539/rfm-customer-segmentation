# Customer Segmentation using RFM Analysis

## 📊 Overview
This project performs customer segmentation using RFM (Recency, Frequency, Monetary) analysis.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib

## 🔄 Steps
- Data Cleaning
- Feature Engineering
- RFM Calculation
- RFM Scoring (pd.qcut)
- Customer Segmentation
- Visualization
- Insights

## 📈 Key Insights
- Majority of customers are in At Risk segment
- Loyal customers are fewer
- Some customers are already lost(low engagement)
- Business should focus on retention strategies

## 📁 Files
- RFM_Customer_Segmentation.ipynb → Full project code
- RFM_Final_Output.csv → Final output dataset

## 🎯 Conclusion
RFM analysis helps understand customer behavior and supports data-driven decisions for improving customer retention and business growth.
